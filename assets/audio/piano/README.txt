ピアノ音源の配置場所

このフォルダに個別のピアノ音源（MP3形式）を配置してください。

ファイル名ルール：
- 半角英数字
- シャープは s で表現（例: Cs4.mp3, Fs3.mp3）
- オクターブ番号付き（例: C3.mp3, D4.mp3）

推奨範囲：C3〜B5（36音）

音源がない場合、Web Audio API による代替音（シンセサイザー音）が使用されます。

例：
  C3.mp3
  Cs3.mp3
  D3.mp3
  Ds3.mp3
  E3.mp3
  F3.mp3
  Fs3.mp3
  G3.mp3
  Gs3.mp3
  A3.mp3
  As3.mp3
  B3.mp3
  ...
